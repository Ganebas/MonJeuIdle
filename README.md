# 🌌 Mon Jeu Idle : Le Mendiant Galactique

Bienvenue sur le dépôt officiel de **Mon Jeu Idle**, un jeu incrémental complet et immersif développé de A à Z en pur HTML, CSS et JavaScript (Vanilla), sans aucun moteur de jeu externe !

🎮 **[👉 JOUER AU JEU MAINTENANT 👈](https://Ganebas.github.io/MonJeuIdle/)**

---

## 🚀 À propos du jeu
Partez de zéro en tant que simple Mendiant Galactique et cliquez pour amasser vos premières ressources. Achetez des bâtiments, optimisez votre production et débloquez des mécaniques de jeu de plus en plus complexes pour conquérir la Galaxie !

## ✨ Fonctionnalités Principales

* **Économie Incrémentale :** Des dizaines de bâtiments et d'améliorations pour automatiser la production.
* **Système de Prestige (Ascension) :** Recommencez avec des bonus multiplicateurs et récoltez de la Matière Noire pour acheter des améliorations cosmiques.
* **Arbre de Compétences RPG :** Dépensez vos points de prestige pour personnaliser votre style de jeu.
* **Mini-jeux intégrés :** * 📈 **La Bourse :** Achetez et revendez des actions dont le prix fluctue en temps réel.
  * ⚔️ **La Tour Sans Fin :** Combattez des boss de plus en plus puissants avec un système de chronomètre.
  * 🎰 **Le Casino :** Tentez votre chance à la machine à sous ou avec la Loterie Quotidienne.
* **Événements Dynamiques :** Attrapez les bonus dorés volatiles ou sauvez vos ressources des pluies de météorites !
* **Système de Familiers :** Incubez des œufs mystères pour obtenir des familiers passifs.
* **Quêtes Journalières :** Des objectifs qui se réinitialisent toutes les 24 heures pour gagner des gemmes.
* **Sauvegarde Avancée :** Sauvegarde locale automatique et système d'Export/Import par code pour partager sa partie.
* **Audio Dynamique :** Bruitages intégrés (clics, achats, succès) et musique de fond.

---

## 🛠️ Technologies Utilisées
Ce projet est un défi de programmation "Vanilla" :
* **HTML5** (Structure et interface)
* **CSS3** (Design néon/sombre, animations, flexbox/grid)
* **JavaScript (ES6)** (Logique de jeu, boucle de rendu, gestion du temps, manipulations du DOM, Web Audio API)

---

## 💻 Installation en local (Pour les développeurs)
Si vous souhaitez télécharger le code et le modifier :
1. Clonez ce dépôt : `git clone https://github.com/Ganebas/MonJeuIdle.git`
2. Ouvrez le dossier dans votre éditeur de code (ex: VS Code).
3. Lancez le fichier `index.html` dans n'importe quel navigateur web moderne.
*(Aucun serveur ou base de données complexe n'est requis !)*
